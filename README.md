# POUR 솔루션 CRM 웹앱

## 프로젝트 개요
POUR 솔루션의 현장 관리 및 견적 문의 CRM 시스템입니다.
모바일 우선(Mobile-First) 설계로 Android/iOS 앱스토어 등록을 목표로 합니다.

## 주요 기능

### 1. 신규 견적 문의 관리
- 잔디 메신저 연동으로 새로운 견적 문의 자동 등록
- 현장명과 공사유형이 버튼으로 표시
- 클릭 시 전체 견적 내용 팝업으로 확인

### 2. 담당자 배정
- 드래그 앤 드롭으로 담당자에게 업무 배정
- 담당자 목록 페이지네이션 (한 페이지 3명)
- 잔디 메신저 담당자 배정 알림 시 자동 배정

### 3. 업무 체크리스트
8가지 서비스 항목 체크 관리:
1. POUR공법 기술 서비스 제공
2. AI 기반 하자 진단
3. 교육 및 가이드 제공
4. 설계도서 / 견적서 제공
5. 기술연구원 관리 감독
6. 전-중-후 PT 진행
7. 행정 업무 지원
8. 현장 설명회 진행

### 4. 현장 상세 관리
- 사진 촬영/갤러리 업로드 (최대 3장)
- PT 자료(PPT) 업로드
- 메모 입력
- 완료/중단 처리 (중단 시 사유 필수 입력)

### 5. 하단 네비게이션
- **홈**: 메인 대시보드
- **일정**: 담당자별 일정 캘린더
- **자료**: 유형별 폴더 분류 갤러리

## 기술 스택

### Frontend
- HTML5, CSS3 (Mobile-First)
- Vanilla JavaScript (ES6+)
- CSS Grid & Flexbox
- Touch Events & Drag-and-Drop API

### Backend
- Node.js + Express
- JSON 파일 기반 데이터 저장 (추후 DB 연동)
- 잔디(Jandi) API 연동

## 실행 방법

### 백엔드 서버 실행
```bash
cd backend
npm install
npm start
```

### 프론트엔드 실행
`frontend/index.html` 파일을 브라우저에서 열거나, Live Server 사용

## 잔디 API 연동
잔디 Webhook URL을 통해 새로운 견적 문의 메시지를 수신합니다.
설정 방법은 `docs/API.md` 참조
